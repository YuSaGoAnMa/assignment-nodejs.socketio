# assignment-nodejs.socketio
## Tutorial
```bash
#clone repository
git clone https://github.com/YuSaGoAnMa/assignment.nodejs.socketio

#change directory
cd assignment.nodejs.socketio

#install dependencies
npm install

#transpile typescript
npm run tsc

#run everything
npm start

#start your browser!
#http://localhost:8080/
```
